# cooking-lcy-vue2

> 一个cooking插件扩展vue2配置.
> 集成less, sass, postcss, 也可以支持stylus和styl, 不过loader请自行安装

使用方法:
```
{
  extends: ['lcy-vue2', 'eslint'] // 使用lcy-vue2后无需再用less, sass
}
```

## include
- "vue-loader": "^9.5.0",
- "less": "^2.7.1",
- "less-loader": "^2.2.3",
- "node-sass": "^3.4.2",
- "sass-loader": "^3.1.2"
